# cio

A C-style I/O wrapper for C++.

> [!NOTE]
> C++17 is required.

## Installation

### Swift Package Manager

Add a package dependency to https://github.com/sbooth/cio in Xcode.

### Manual or Custom Build

1. Clone the [cio](https://github.com/sbooth/cio) repository.
2. `swift build`.
